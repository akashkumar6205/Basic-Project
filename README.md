# Calculator



A simple, elegant web-based calculator application built with HTML, CSS, and JavaScript.

## Project Overview

This project is a fully functional calculator that allows users to perform basic arithmetic operations with a clean and modern user interface. The calculator features a dark-themed design with a gradient background and intuitive button layout.

## Features

- **Basic Arithmetic Operations**: Supports addition (+), subtraction (-), multiplication (*), and division (/)
- **Percentage Calculation**: Calculate percentages using the % operator
- **Clear Function**: AC (All Clear) button to reset the calculator
- **Delete Function**: DEL button to remove the last entered digit
- **Decimal Support**: Ability to work with decimal numbers
- **Responsive Design**: Clean, centered layout with gradient background
- **Double Zero**: Quick entry of 00 for convenience

## Files Structure

- `index.html` - Main HTML structure defining the calculator layout
- `app.js` - JavaScript logic for calculator functionality and button interactions
- `style.css` - CSS styling with modern design and dark theme

## How to Use

1. Open `index.html` in your web browser
2. Click on number buttons (0-9) to enter values
3. Select an operation (+, -, *, /, %)
4. Click the equals (=) button to calculate the result
5. Use AC to clear all entries or DEL to remove the last digit

## Styling Details

- Dark gradient background (black to dark blue-gray)
- Modern Poppins and Montserrat fonts from Google Fonts
- Transparent calculator container with subtle shadow effect
- Responsive button layout organized in rows

## Installation

No installation required! Simply download or clone the files and open `index.html` in any modern web browser.

## Browser Compatibility

Works on all modern browsers that support:
- ES6 JavaScript
- CSS Grid/Flexbox
- HTML5
