# Project Portfolio

This workspace contains several small web projects built with HTML, CSS, and JavaScript. Each project is self-contained and can be opened directly in a browser.

## Projects

### 1. Analog Clock
- Description: A web-based analog clock showing the current time with smoothly moving hour, minute, and second hands.
- Files: `Analog colck/index.html`, `Analog colck/style.css`, `Analog colck/app.js`
- Run: Open `Analog colck/index.html` in a browser.
- Notes: Uses vanilla JavaScript and CSS to animate the clock hands in real time.

### 2. Calculator
- Description: A dark-themed calculator that supports addition, subtraction, multiplication, division, percentage, decimal values, delete, and clear operations.
- Files: `claculator/index.html`, `claculator/style.css`, `claculator/app.js`
- Run: Open `claculator/index.html` in a browser.
- Notes: Designed for everyday calculations with a clean responsive interface.

### 3. Email Validator
- Description: A client-side email validation form that checks email format and provides instant user feedback.
- Files: `Email validator/index.html`, `Email validator/css/style.css`, `Email validator/js/index.js`, `Email validator/img/`
- Run: Open `Email validator/index.html` in a browser.
- Notes: Validation is performed in the browser; production apps should also include server-side validation.

### 4. Photography Project
- Description: A simple photography portfolio layout for showcasing images and photo projects.
- Files: `photography/index.html`, `photography/style.css`
- Run: Open `photography/index.html` in a browser.
- Notes: Add image assets and edit the markup to customize the portfolio presentation.

### 5. Random Color Generator
- Description: Generates random colors in the browser and updates the page styling dynamically.
- Files: `random color generator/index.html`, `random color generator/style.css`, `random color generator/app.js`
- Run: Open `random color generator/index.html` in a browser.
- Notes: Demonstrates DOM manipulation, event handling, and dynamic styling.

### 6. Simon Says Game
- Description: A browser-based Simon Says game where the player repeats an increasing sequence of button presses.
- Files: `Simon Says Game/index.html`, `Simon Says Game/style.css`, `Simon Says Game/app.js`
- Run: Open `Simon Says Game/index.html` in a browser and press any key to start.
- Notes: Uses colored buttons and game logic to track sequence and player input.

### 7. Spotify Clone
- Description: A static frontend mockup of the Spotify web UI created for practice and UI styling.
- Files: `Spotify clone/index.html`, `Spotify clone/asset.html`, `Spotify clone/style.css`, `Spotify clone/asset_files/`
- Run: Open `Spotify clone/index.html` in a browser.
- Notes: Visual-only clone; it does not play music or include streaming functionality.

### 8. To Do List
- Description: A simple client-side task list app for adding, completing, and removing tasks.
- Files: `To Do List/index.html`, `To Do List/style.css`, `To Do List/script.js`
- Run: Open `To Do List/index.html` in a browser.
- Notes: No persistence is included; tasks are stored only in memory while the page is open.

## How to Use
1. Open the `project` folder in your file explorer.
2. Select the project folder you want to view.
3. Open the `index.html` file in a modern browser.

## Browser Compatibility
All projects are built with standard HTML, CSS, and JavaScript and should work in modern browsers such as Chrome, Edge, Firefox, and Safari.

## Notes
- No installation or build steps are required for these projects.
- Each app is static and runs locally from the browser.
- If you want to use a local server, run a simple HTTP server from the project folder.
