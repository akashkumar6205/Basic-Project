# Email Validator

A simple client-side Email Validator web project that validates email input using JavaScript. This project demonstrates basic form validation techniques, DOM manipulation, and user-friendly feedback using HTML, CSS, and JavaScript.

## Features
- Real-time email format validation
- Clear visual feedback for valid/invalid input
- Lightweight and easy to integrate into other projects

## How to use
1. Open `index.html` in your browser.
2. Enter an email address in the input field.
3. The validator will indicate whether the email format is valid.

## Project structure
- `index.html` — main HTML file
- `css/style.css` — styles
- `js/index.js` — JavaScript logic for validation
- `img/` — images used by the UI

## Notes
- Validation is performed on the client side; consider adding server-side validation for production use.
